# Toy Project : BlindDate ( 2020.12.24 ~ ing )

## Process
* 1차 : 아이디어 회의
* 2차 : 아이디어 상세회의
* 3차 : 개발 단계 세분화 회의 ( 1 ~ n 단계 )
* 4차 : 세분화 각 단계에 해당하는 부분 설계
* 5차 : 개발 진행
* 6차 : 프로젝트 회고 ([KPT 방법론](https://woowabros.github.io/experience/2020/05/13/birth-of-team-culture.html))
